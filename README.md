## Command list

##### Command Prefix: |

#### @EVERYONE
* help command: `help`
* join voice channel: `join`
* plays a YouTube video: `play [YouTube link]`
* skips current queue'd song: `skip`

#### @ADMINISTRATOR
* kicks mentioned user: `kick [@user]`
